BI Project â€“ Streamlit Cost Estimation App for Travel Itineraries

This project consists of an intelligent cost estimator that allows travel agents and clients to generate instant price quotes based on selected services. Built entirely in Python and Streamlit, this web app connects to a centralized Google Sheet that contains structured pricing data and rules.

Problem Description

The challenge addressed in this project was the manual and time-consuming process of calculating costs for travel packages. Pricing data was scattered across multiple Excel and Google Sheets, with varying formats and conditions based on group size and service type (per person vs. group pricing).

Problem Resolution

I created a centralized and automated pricing app using Streamlit. It connects to a Google Sheet with real-time cost data, differentiates between services charged per person and per group, and dynamically applies pricing rules based on group sizes. The tool calculates total quotes in seconds, removing the need for error-prone manual computation.

Key Features

  • Dynamic form with day-by-day service selection
  • Automatic cost calculation based on pricing logic
  • Integration with centralized Google Sheet as source of truth
  • Final summary table and total cost calculation
  • Built with Python + Streamlit, deployed on Streamlit Cloud

Tools Used

  • Python
  • Streamlit
  • Google Sheets (API)
  • Pandas
